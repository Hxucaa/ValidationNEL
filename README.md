## Installation

### CocoaPods

Simply add the following line to your Podfile:

```ruby
pod "ValidationNEL"
```

And then run:

```console
pod install
```

### Carthage

Just add the following to your [`Cartfile`](https://github.com/Carthage/Carthage/blob/master/Documentation/Artifacts.md#cartfile)

```swift
github "Hxucaa/ValidationNEL"
```

And run:

```console
carthage update
```
